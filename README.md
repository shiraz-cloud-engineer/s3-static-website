# 🌐 Static Website Hosting on AWS S3

Personal portfolio website hosted on Amazon S3.

## 🔗 Live URL
http://shiraz-static-website-2026.s3-website-us-east-1.amazonaws.com

## 🛠 Services Used
- Amazon S3 — Static website hosting
- S3 Bucket Policy — Public read access
- AWS Console — Deployment

## 📋 Steps to Deploy
1. Create S3 bucket
2. Enable static website hosting
3. Upload index.html and error.html
4. Set bucket policy for public access
5. Access via S3 website endpoint URL
